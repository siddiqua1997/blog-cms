## Content Framework for Every Post

Each blog article must follow this SEO structure:

### Required Elements

1. **Primary Keyword Placement**
   - [ ] In the title (H1)
   - [ ] In the first paragraph (within first 100 words)
   - [ ] In at least one H2 subheading
   - [ ] In the meta description
   - [ ] In the URL slug
   - [ ] In image alt text

2. **Secondary Keywords**
   - Include 3-5 related keywords naturally throughout
   - Use semantic variations (e.g., "remap" / "remapping" / "tune")

3. **Content Structure**
   ```
   H1: Primary Keyword in Title

   Introduction (include primary keyword)

   H2: What is [Topic]?
   H2: Benefits of [Topic]
   H2: [Topic] Process Explained
   H2: Cost and Pricing
   H2: FAQs About [Topic]

   Conclusion with CTA
   ```

4. **Internal Linking**
   - Link to 2-3 related articles
   - Link to service pages when relevant
   - Use descriptive anchor text (not "click here")

5. **Media Requirements**
   - Hero image with keyword-rich alt text
   - At least 1-2 supporting images
   - Consider video embeds for complex topics

## SEO Writing Rules

### Structure
- Use short paragraphs (2-3 sentences max)
- Use bullet points and numbered lists
- Include H2 and H3 subheadings every 200-300 words
- Aim for 1,500-2,500 words for pillar content
- 800-1,200 words for standard posts

### Readability
- Write at 8th-grade reading level
- Use active voice
- Avoid jargon without explanation
- Define technical terms on first use

### On-Page SEO
- Meta title: 50-60 characters
- Meta description: 150-160 characters
- URL slug: lowercase, hyphenated, keyword-focused
- Image alt text: descriptive, includes keyword when natural

### Trust Signals
- Include specific data and statistics
- Reference dyno results when available
- Mention certifications and experience
- Add customer testimonials where relevant
